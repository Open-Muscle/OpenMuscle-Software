Before you tap Enter VR

  • HTTPS or localhost
  • WebXR + hand-tracking available
  • Server reachable (/api/devices)
The Quest hand on this side is the one whose joints we record. The other hand stays free for the record button.

If "Enter VR" stays disabled, the headset can't see WebXR on this URL. Check that you opened it in Quest Browser (not the Library / Phone-mirror window), and that the URL is HTTPS or http://localhost (via adb reverse tcp:8000 tcp:8000).
