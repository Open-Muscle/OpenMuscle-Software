```bash
mpremote mip install github:jposada202020/MicroPython_BMI160
mpremote mip install ssd1306

```

## Pinouts

### I2C Pins
- SCL: 9
- SDA: 8

### Buttons

- Menu : 11
- Select : 10
- Joystick Push Button : 7

### Joystick Analog Pins

- Joystick X-Axis : 6
- Joystick Y-Axis : 5
